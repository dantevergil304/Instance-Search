All data is organized as 244 directories (named from video0 to video243). Each directory will have subdirectories which are its corresponding shots.
This directory including:
- all_frames: all the frames extracted from shots of 244 videos.
- all_faces: all the faces detected using MTCNN from frames.
- all_faces_python: all the faces save as python2 pickle.
- all_landmarks: all the 5-pts landmarks from MTCNN.
